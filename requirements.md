Goal is to simplify docker install of openclaw by providing copy-pasteable bash script on windows (git bash), linux or mac.

- A HTML + CSS + JS based UI
- Refer to https://docs.openclaw.ai/install/docker and design a wizard kind of ui, where the user is presented with options (text boxes if he needs to enter his specific stuff)

Finally, the ui shows a shell script code block according to the user's os
