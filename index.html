Your Installation Script

Copy and paste this script into your terminal to install OpenClaw with Docker.

install-openclaw.sh 
#!/bin/bash

# OpenClaw Docker Installation Script
# Generated by OpenClaw Docker Setup Wizard

set -e

echo "Starting OpenClaw Docker installation..."

# Clone repository
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# Run setup script
./docker-setup.sh

echo "Installation complete!"
echo "Open http://127.0.0.1:18789/ in your browser"
echo "Use the token shown above to authenticate"

Next Steps

  • Open http://127.0.0.1:18789/ in your browser
  • Paste the generated token into the Control UI (Settings → Token)
  • Run docker compose run --rm openclaw-cli dashboard --no-open to get the dashboard URL anytime
